# minimarket

 # LOGIN

 1) -- > Se inicia una ventana la cual si el programa no se abrio antes y se registro antes al menos un administrador, va a abrirse en la ventana para registrar un administrador, al registrarlo se lo va a guiar al login, donde debe elegir el tipo de usuario(usuario/admin) para iniciar, su nombre y contrasenia.

2) -- > Para el programa puede registrarse tanto usuarios como administradores quiera, se recomienda un solo administrador, y para poder crear otro debe ingresar un codigo el cual va a permitir tener mas de 1 administrador(metodo de seguridad para que no cualquiera que este en frente de el programa pueda crearse una cuenta administrador y manipular los datos del mismo); Para poder crear mas administradores debe ingresar el codigo "-1"(menos uno) el cual va a permitir crear otro administrador.

3) -- > Ademas puede recuperar su cuenta si se olvida su contrasenia, el unico requisito es acordarse el id de la cuenta el cual se lo muestra abajo en el centro apenas se abre el programa(lo muestra los primeros 5 segundos al abrirse).


# INICIO 

1) -- > Al iniciar el programa luego de el login se va a presentar una ventana con 3 subventanas a su izquierda y botones. La primera ventana se llama "Datos" la cual es el centro de administraciond e el programa, ahi va a poder manipular todos los datos que su programa va a manejar, productos, proveedores, categorias(borrado, edicion y visualizacion de cada uno de ellos) e incluye un boton de Borrar Datos el cual borra toda la base de datos tanto logins como registros.

2) -- > Ventana Busar Datos, en esta ventana encontraremos los resumenes tanto del día como de el mes, va a poder ver ventas, compras y detalles de cada una de ellas.

3) -- > Ventana Administracion la cual vamos a poder facturar y hacer compras para nuestro negocio, en la cual en tiempo real se va a actualizar y ver los productos por detras de el facturero o compra(lugar donde se registran las ventas y compras).


-- > Para el tipo de usuario "administrador" se van a ver las 3 ventanas y para el tipo de usuario "usuario" solo se vera Buscar datos y Administracion.
